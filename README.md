# スロット崩し (Slot Breaker)

## 概要

スロットとブロック崩しを組み合わせた認知症予防ゲームです。

プレイヤーはメダルを使ってスロットを回し、揃った絵柄に応じて特殊能力を持つボールを獲得します。

発射装置（ランチャー）は自動で動き続けるため、プレイヤーはタイミングを見極めるだけで遊べます。

高齢者・中年層・若年層まで幅広い世代が楽しめることを目標に開発しています。

---

## コンセプト

### 目的

**「毎日5分でも遊びたくなるゲーム」**

ゲームを楽しみながら自然に以下の能力を使うことを目指します。

* 色の認識
* 状況判断
* 短時間の集中
* 成功体験の積み重ね

---

## ゲームの流れ

1. メダルを消費してスロットを回す
2. 絵柄が3つ揃う
3. 対応するボールを獲得
4. ランチャーから自動発射
5. ブロックを破壊
6. メダル獲得エリアで報酬獲得
7. 再びスロットを回す

このサイクルを繰り返してプレイします。

## ランチャーシステム

* 30°～150°の範囲を自動往復
* 初期角度90°
* 一定速度で回転
* プレイヤーは操作不可
* スロット成立時に自動発射

プレイヤーはタイミングを見極めることに集中します。

* メダルエリア拡大
